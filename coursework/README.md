# Course Restudy Workspace

Use this folder while restudying the AWS Generative AI and AI Agents with Amazon Bedrock Professional Certificate.

The goal is to keep rough learning work organized without diluting the polished `labs/`, `src/`, `docs/`, and `notes/` sections.

## Structure

- `course-01-getting-started/` - getting started with AWS GenAI for developers.
- `course-02-generative-ai-applications/` - RAG, knowledge bases, agents, and application patterns.
- `course-03-customization-optimization-automation/` - evaluation, optimization, automation, and advanced implementation patterns.

## Rules

- Write your own code and notes.
- Do not copy Coursera quizzes, proprietary exercise text, or paid course material.
- Move polished, reusable code into `src/bedrock_labs/`.
- Move polished demo labs into `labs/`.
- Keep credentials, AWS profiles, local outputs, and private data out of Git.

## Workflow

1. Add rough code while studying inside the relevant `coursework/course-*` folder.
2. When an exercise becomes portfolio-grade, refactor reusable code into `src/bedrock_labs/`.
3. Add a polished demo README under `labs/`.
4. Update `CERTIFICATE_ROADMAP.md` and the root `README.md` with progress.
